# Canada-Crime-Statistics-Dashboard
This is an interactive Power BI dashboard to analyze crime trends across Canada, providing insights for policymakers and law enforcement. The dashboard visualizes Year-over-year crime trends, Category-wise and geographic crime breakdowns, Crime hotspot mapping for better resource allocation, etc

The dashboard goes beyond basic charts, offering dynamic analysis and user-friendly exploration, including:

1. Drill-through – click visuals to see detailed records
2. Page navigation – switch seamlessly between summary and detail pages
3. AI Q&A – ask natural-language questions and get instant answers
4. Decomposition tree – break down crime metrics by multiple factors

Key analysis includes:

1. Year-over-year crime trends
2. Category-wise and geographic breakdowns
3. Crime hotspot mapping to support better resource allocation
